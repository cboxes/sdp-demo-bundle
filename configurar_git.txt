1. Verificar si Git ya está instalado
git --version

Si aparece algo como:

git version 2.50.1.windows.1

ya está instalado.


2. Descargar Git para Windows

Ingresa a la página oficial: https://git-scm.com/downloads

3. Instalar Git

Durante la instalación puedes dejar casi todo por defecto.

Recomendaciones:

Selección de editor

Cuando aparezca:

Choosing the default editor

Selecciona:

Use Visual Studio Code as Git's default editor

HTTPS

Selecciona:

Use the OpenSSL library

Terminal

Selecciona:

Use Windows' default console window

o

Use Git Bash

Ambas funcionan.


5. Configurar tu identidad Git

Configura tu nombre:

git config --global user.name "Carlos Cajas"

Configura tu correo:

git config --global user.email "tu_correo@gmail.com"

Verifica:

git config --list

Debes ver:

user.name=Carlos Cajas
user.email=tu_correo@gmail.com


6. Instalar extensión Git en VS Code

En VS Code:

Extensions

Busca:

GitHub Pull Requests

o

GitLens

Recomendado para Data Engineering:

GitLens
GitHub Pull Requests

7. Crear repositorio para tu proyecto Databricks

En tu caso:

sdp-demo-bundle

Desde VS Code:

cd C:\Users\Carlos\Desktop\sdp-demo-bundle

Inicializa Git:

git init

Resultado:

Initialized empty Git repository
